## English

A modern, professional timer and clock application with multiple visual themes and comprehensive customization options. Perfect for presentations, workouts, meetings, training exercises, emergency preparedness, crisis management, and any scenario requiring precise time management and operational robustness.

> **🎯 Ready to use?** [**Try the live demo at mackan.eu/timer**](https://mackan.eu/timer) - No installation required!

### ✨ Features

#### **Visual Themes**
- **Digital Standard** - Clean, modern digital display
- **Analog Station** - Classic Swiss railway station clock
- **7-Segment Display** - Retro digital style
- **Tactical** - Military/tactical themed display
- **Nixie Tubes** - Vintage electronic tube aesthetic
- **LCD Display** - Classic LCD screen appearance
- **Split-Flap Display** - Mechanical airport/train station style
- **LED 7-Segment** - Bright LED segment display

#### **Functionality**
- **Timer Mode**: Countdown and count-up timers
- **Clock Mode**: Real-time and custom time display
- **Exercise Details**: Custom titles and preparation phases
- **Progress Indicators**: Visual progress bars and circle charts
- **Time Manipulation**: Speed control (0.5x to 60x speed)
- **Fullscreen Support**: Immersive fullscreen experience
- **Display Scaling**: Adjustable size (25% - 100%)

#### **Customization**
- **Bilingual Interface**: Swedish and English support
- **Theme Switching**: Light and dark modes
- **Custom Messages**: Personalized exercise titles and pause messages
- **Flexible Timing**: Hours, minutes, seconds configuration
- **Date Display**: Optional date and weekday showing

#### **Technical Features**
- **Offline Support**: Works without internet connection
- **Responsive Design**: Adapts to different screen sizes
- **Keyboard Shortcuts**: Space (pause/resume), R (restart), Esc (exit fullscreen)
- **Local Storage**: Saves all settings and preferences
- **Modern Web Technologies**: HTML5, CSS3, JavaScript ES6+

### 🚀 Quick Start

**Option 1: Try Online (Recommended)**
1. **Visit**: [mackan.eu/timer](https://mackan.eu/timer) 
2. **Configure**: Set your desired time and theme
3. **Start**: Click "Start in Fullscreen" or press F11

**Option 2: Run Locally**
1. **Download**: Get the latest release or clone this repository
2. **Open**: Launch `index.html` in any modern web browser
3. **Configure**: Set your desired time and theme
4. **Start**: Click "Start in Fullscreen" or press F11
